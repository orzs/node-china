# node-china
a community which running on node.js.

## requirements

- node.js v0.10.33+
- mongodb v2.6.4+
- redis  v3.0.2+
- elasticsearch v2.1.0+
- bower v1.4.1+
- GNU make v3.8.1+

## start
```
  $ npm install 
  $ cd public & bower install
  $ make start
```
